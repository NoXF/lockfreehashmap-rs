pub struct AtomicPtr<T>(Atomic<T>);

Wrapper around [Atomic]. Represents a pointer that can be null at first but never again afterwards.

Swaps out the current value, leaving null in its place. Equivalent to [Option::take()].

Unsafe

This is unsafe because there must be no other pointers that can ever access this AtomicPtr after this function is called. This is intended to be used for [Drop::drop()].
